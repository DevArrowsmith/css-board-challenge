Selectors

  • Attribute Selectors

    Attribute selectors target elements based on attributes they contain i.e. "href", "type" etc.

  • Pseudo-Element Selectors

    Pseudo-Element Selectors allow you to target specific parts of an element without having to name them in advance - for instance, p::first-line which will target only the first line of the element

  • Class

    A CSS class is an attribute used to define a group of HTML elements in order to apply unique styling and formatting to those elements with CSS.

  • Element Selector

    The element selector selects all elements with the specified element name.

Mixed

  • Borders

    Borders are implemented as you guessed it, when you want to add a border to some paragraph, header etc.

  • Z-Index

    Z Index (z-index) is a CSS property that defines the order of overlapping HTML elements. Elements with a higher index will be placed on top of elements with a lower index.

  • Margin

    The space outside of a border surrounding an element

  • Padding

    used to generate space around an element's content, inside of any defined borders.

  • Box-Shadow

    adding a coloured shadow behind an element

  • Overflow

    Sets the behaviour for content which is too large for its container - i.e. should it spill out to the right, or to the bottom, etc.

  • Absolute Positioning

    An element with position:absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).

  • Position: Sticky

    An element with this attribute will have a postion of relative until it reaches a specific scrolling threshold. It will then behave as a fixed element.
